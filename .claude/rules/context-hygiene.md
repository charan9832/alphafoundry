# Context Hygiene Rule

AlphaFoundry work should preserve focused context.

Rules:
- Start fresh sessions for implementation after planning.
- Use `/af-prime` at the beginning of a fresh task session.
- Use subagents/reviewers for broad review instead of loading everything into the main context.
- Do not read the entire repo blindly.
- Prefer specific files and recent git history.
- Use `/af-context-budget` before long work or when context gets noisy.
- Use compact summaries at phase boundaries: planning -> implementation -> review.
