# Before Coding Checklist

- Read `AGENTS.md` and `CLAUDE.md`.
- Check `git status --short --branch`.
- Identify exact task/slice.
- Identify allowed files.
- Identify tests to add/update first.
- Confirm no finance boundary violation.
- Confirm Pi substrate reuse; no native tool execution duplication.
- Confirm verification commands.
- If task is ambiguous, ask questions before editing.
