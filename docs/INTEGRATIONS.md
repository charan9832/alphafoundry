# Integrations Roadmap

## Required/near-term
- OpenBB for financial data.
- SimpleMem for agent memory.
- LLM Wiki as global research brain.
- Autoresearch loop for strategy improvement.
- vectorbt for scalable backtesting/parameter sweeps.

## Models
- Kronos: financial K-line foundation model.
- TimesFM: general time-series forecasting baseline.
- MOMENT: representation/anomaly/forecasting tasks.
- Time-MoE: large-scale time-series forecasting.
- FinBERT-style models: sentiment/news later.

## Multi-agent references
- TradingAgents: analyst/researcher/trader/risk roles.
- QuantDinger: self-hosted quant OS inspiration.
